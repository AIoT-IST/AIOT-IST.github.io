Script Setting

  1. Time Stop: Wait the time after executing each instruction.
  2. Spt Step Speed: It is the minimum waiting time between instructions.

Example : Time Stop

Use the both setting.

Look at the below figure that the second instruction adds 3s than the first instruction.

How to get the 3s ?

  • “Time stop” is 1000ms
  • “Sleep” is 3
  • 3s = Time stop * Sleep

Example : Spt Step Speed

Use the setting that the “Spt Step Speed” is 1000ms.

Watch that the second instruction is running delay 1s after the first instruction.

Scripts Managment

Provide Multi-Script Management to define the trigger command table.

In other words, You can use RESTFUL or MSMQ to trigger the required script.

Open the MSMQ Execute Scripts Management

Path ==> Tools ==> MSMQ Execute Scripts Management

1. Choose the MSMQ Cmd

2. Select the Scripts file

3. Click the Change.

AutoRun

Run the Background Schedule setting

Path : Tool ==> Config ==> Bakground Schedule Setting

Select the “Script File” that you want to run your design script.

Note: Enable the “Run script if HID keeps idling over timeout.”

You also can click the Start(Auto) without select the “.spt” again when selected the script file.

HID idle time: AutoRun the script when both keyboard & mouse are idle over the set time.

Autorun screensaver timeout: The 10 seconds will count down before automatic execution
